## 🔍 Plan Details & Limits

### 1. ChatGPT (OpenAI)
* **Plus ($20/mo)**: Designed for single users. Standard quotas for GPT-4o, Custom GPT access, and advanced voice. Data is used for training unless you manually opt out in settings.
* **Team ($30/mo)**: *Requires a 2-seat minimum*. Shared team workspace, admin console, and higher usage limits. **Data is not used for model training by default**.
* **Enterprise (~$60/mo)**: *Requires a 150-seat minimum or custom sales contract*. SSO, team usage analytics, priority support, and infinite message caps.

### 2. Claude (Anthropic)
* **Pro ($20/mo)**: High-limit access to Claude 3.5 Sonnet. Access to Projects (context sharing).
* **Team ($30/mo)**: *Requires a 5-seat minimum*. Higher usage caps (5x Pro), administrative billing, and workspace data privacy settings.
* **Enterprise (~$75/mo)**: Custom annual contract. Advanced data compliance, SAML SSO, and dedicated account support.

### 3. Cursor (Anysphere)
* **Pro ($20/mo)**: 500 fast premium requests per month for Claude 3.5 Sonnet / GPT-4o, unlimited slow queries, and unlimited Cursor autocomplete.
* **Business ($40/mo)**: Admin provisioning, SSO, privacy mode enabled by default (zero code data saved on Cursor servers), and shared team keys.
* **Enterprise (~$80/mo)**: Custom contract. IP indemnity protection and custom firewalls.

### 4. GitHub Copilot (Microsoft)
* **Individual ($10/mo)**: Basic code autocomplete and chat integration in standard IDEs (VS Code/JetBrains).
* **Business ($19/mo)**: Shared billing, organization policy controls, and intellectual property indemnity.
* **Enterprise ($39/mo)**: Custom repository-level indexing, documentation search, and advanced CLI debugging.
